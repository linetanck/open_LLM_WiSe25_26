Projekt für das Seminar Open LLMs in der Praxis.

## Aufgabenstellung:
Open LLMs nutzen, um kurze Texte (Songlyrics) auf einen Bestimmten Aspekt, wie die Einstellung von dem lyrischen Ich zu dem lyrischen Du rauszufinden. 
Mögliche Kategorien könnten sein: "positiv", "negativ" oder "glücklich verliebt", "unglücklich verliebt", "getrennt"
